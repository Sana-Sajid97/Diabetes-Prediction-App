# Diabetes-Prediction-App

This is a simple Streamlit app that predicts whether a person is likely to have diabetes based on medical inputs.

## Features

- Input your age, glucose level, blood pressure, BMI, and insulin level  
- Predicts if a person may have diabetes or not  
- Interactive and easy to use

## How to run locally

1. Clone this repository  
2. Make sure you have `app.py` and `diabetes_model.pkl`  
3. Install dependencies:  
   ```bash
   pip install streamlit
   
   streamlit run app.py
